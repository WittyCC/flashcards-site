Flashcards

  • JavaScript
  • Object-oriented computer programming language, commonly used to create interactive effects within web browsers.
  • Operators
  • Symbol used to perform an operation (+, -, %, etc.;)
  • Variables
  • Named "containers" where you can place data and refer to it later.
  • Variables Naming Conventions
  • Styling conventions, such as two-space indentation and brace location, as well as minimizing globals by using 'var'.
  • Functions
  • A tool that can be used repeatedly, taking in values and returning a value.
  • Methods
  • Actions performed on objects, containing a function definition.
  • Arguments vs Parameters
  • Parameter is the variable name, argument is the user input.
  • Return
  • Return statement stops the function, and returns a value.
  • Chaining methods
  • a.k.a 'Cascading', applying more than one method to an object.
  • Strings, Booleans, Numbers
  • 3 common data types. Different methods can be called, depending on type.
  • Undefined
  • Another data type, only possible value of 'undefined'. Javascript initially declares variables' values as undefined.
  • NaN
  • "Not a Number" - still considered a number.
  • Escape
  • Encodes a string. Now called 'encodeURI()'.
  • Alert
  • Method that displays an alert box with a specified message and an OK button; often used if you want to make sure information comes through to the user.
  • Comments
  • Used to explain JavaScript code and make it more readable.
  • jQuery
  • A JavaScript library used to add extra functionality to websites. It has become the most popular JavaScript library used in web development.
  • Attributes
  • They provide additional information about HTML elements.
